# Definition of Done

- [ ] Tous les AC de la story sont satisfaits
- [ ] Tests unitaires ajoutes/maj et verts
- [ ] Tests d'integration ajoutes/maj et verts
- [ ] `dotnet format` execute et OK
- [ ] `dotnet build` execute et OK
- [ ] `dotnet test` execute et OK
- [ ] Build global de la solution vert (tous projets)
- [ ] Aucun secret en dur dans le code
- [ ] Liste des fichiers modifies complete dans la story
- [ ] Notes techniques de la story completees
- [ ] Causes racines des incidents corriges documentees
- [ ] Au moins un test de regression ajoute par bug corrige
- [ ] Statut story passe a `review` puis `done` apres validation
