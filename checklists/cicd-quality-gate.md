# CI/CD Quality Gate

- [ ] Build CI vert
- [ ] Tous les tests automatiques passent
- [ ] Couverture de code >= seuil minimal defini
- [ ] Scans securite sans vulnerabilite critique/haute non traitee
- [ ] Verification des secrets (aucune fuite detectee)
- [ ] Artefacts publies et exploitables
- [ ] Rapport de gate archive
